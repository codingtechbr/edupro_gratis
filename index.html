Configure o sistema uma única vez

1Crie a planilha no Google Sheets
Acesse sheets.google.com → Criar planilha em branco → Renomeie para EduPro

Copie o ID da URL: docs.google.com/spreadsheets/d/ID_AQUI/edit
2Configure o Google Apps Script
Na planilha: Extensões → Apps Script
Cole o código do arquivo Code.gs
Troque COLE_O_ID_DA_PLANILHA_AQUI pelo ID copiado
Implantar → Novo Implantação → Aplicativo da Web
Acesso: Qualquer pessoa → Implantar → Copie a URL
